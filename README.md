# Wireshark Network Traffic Analysis

## Overview
This repository demonstrates hands-on experience using Wireshark to analyze network traffic,
detect security threats, and investigate suspicious activity.

## Skills Demonstrated
- Packet analysis (TCP, UDP, HTTP, DNS)
- Network security investigation
- Display filter creation
- Malware traffic identification
- PCAP analysis

## Tools
- Wireshark
- TCP/IP
- Network protocols

## Use Cases
- Incident response
- SOC analyst training
- Network troubleshooting
